Um sistema de banco de dados inclui os seguintes:
- Dados
- Hardware
- Software
- Usuários

**Dados**
- Ser integrados
- Compartilhados
- Persistentes

**Hardware**
- Máquina servidor onde o banco de dados irá ser armazenado

**Software**
- Programa onde irei fazer as consultas e administração dos dados

**Usuários:**
- Programadores de aplicações
- Usuários finais (Acessam o db por meio das aplicações)
